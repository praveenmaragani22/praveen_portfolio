## ✨ About

A fully animated, single-file personal portfolio website built with **pure HTML, CSS, and vanilla JavaScript** — no frameworks, no build tools, no dependencies. Just open it in a browser and it works.

Built to showcase my projects, skills, certifications, and contact information with a premium dark-themed UI.

## 🌐 Custom Domain Setup

This site uses a custom domain via GitHub Pages. To replicate:

**1. Add your domain in GitHub**
Go to your repo → **Settings → Pages → Custom domain** → enter your domain → Save.

**2. Configure DNS with your domain registrar**

| Type | Host | Value |
|------|------|-------|
| `A` | `@` | `185.199.108.153` |
| `A` | `@` | `185.199.109.153` |
| `A` | `@` | `185.199.110.153` |
| `A` | `@` | `185.199.111.153` |
| `CNAME` | `www` | `YOUR-USERNAME.github.io` |

**3. Enable HTTPS**
After DNS propagates (up to 24–48 hrs), check **Enforce HTTPS** in Pages settings.

---

## 🚀 Deploy to GitHub Pages

1. Upload all files to your repository
2. Go to **Settings → Pages**
3. Set **Source** to `Deploy from branch` → `main` → `/ (root)`
4. Click **Save**
5. Your site goes live at `https://YOUR-USERNAME.github.io/REPO-NAME/`
